Personal Projects

  • Some of the code for these projects is not publicly available online. If you have questions about those, just send me an email and I'll do my best to answer promptly. I make no promises that I can answer all questions.
  • Motor Datasheet
    • Designed for NASA JSC engineers to supersede an ineffective Excel database
    • Given a set of operating points (voltage, current, etc.), ranks various motors by efficiency
    • Provides simple interface for inserting, deleting, and editing motor entries
    • Utilizes a RESTful API written in Node.js, HTML, and jQuery
  • FRC Scouting Database
    • Developed for the FIRST Robotics Competition to analyze other teams' performance
    • Fully-enabled database application - allows insertion, deletion, and data analysis
    • Displays custom aggregate datapoints, trendlines, and overall rankings for given metrics
    • RESTful API written with Node.js, HTML, MySQL, exported to MS Excel
  • Music Writer
    • Uses a Markov chain to generate novel MIDI files from patterns found in existing songs
    • Only designed to handle solo piano music to maximize simplicity and functionality
    • Extension of a text writer application written for a class project
    • Implemented in Java, with the MIDI format used for imports and exports
